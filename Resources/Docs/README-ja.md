# Slow Quit Apps - インストールガイド

## 初回セットアップ

### 1. アクセシビリティ権限を付与

このアプリはキーボードショートカットを監視するために**アクセシビリティ権限**が必要です。

**有効にする方法：**
1. アプリを開くと、権限の許可を求められます
2. **「システム設定を開く」** をクリック、または手動で以下に移動：
   - **システム設定 → プライバシーとセキュリティ → アクセシビリティ**
3. リストから **「SlowQuitApps」** を見つけます
4. スイッチを **オン** に切り替えます
5. アプリ設定で **「アプリを再起動」** ボタンをクリック

> ⚠️ リストにアプリが表示されない場合は、**+** ボタンをクリックして、アプリケーションフォルダから手動で追加してください。

### 2. アプリリストへのアクセスを許可（オプション）

**アプリ一覧** タブを初めて開くと、インストール済みアプリへのアクセス許可を求められることがあります。これにより：
- ホワイトリストにアプリを追加（長押しなしで終了できるアプリ）
- 保護対象のアプリを管理

---

## 動作の仕組み

- ⌘Q を設定された時間 **長押し** するとアプリが終了
- 途中で離すと終了をキャンセル
- ホワイトリストのアプリは長押しなしで即座に終了可能

## 便利なヒント

- 💡 **一般** 設定で長押し時間を調整（0.3秒〜3.0秒）
- 💡 頻繁に終了するアプリはホワイトリストに追加
- 💡 アプリはメニューバーで動作
